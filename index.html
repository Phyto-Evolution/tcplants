Getting Started

First Login

You need two things to log in: your Encryption Password (you chose this — it never leaves your browser) and a GitHub Personal Access Token with repo scope.

Get a token at: github.com → Settings → Developer settings → Personal access tokens → Classic → New token → tick repo → Generate

⚠️ Never lose your encryption password. There is no recovery — all note content is encrypted with it. The GitHub repo only stores ciphertext.

Session Behaviour

Your token is stored in sessionStorage only — it disappears when you close the tab. Your password is held in memory only. Every lock or tab close wipes both. This is by design.

Lab Notes

Creating an Entry

Click ＋ New Entry. Fill in Title (required), Experiment Type, TC Stage, Species/Accession, Tags, and optionally a Next Review Date for reminders. The Experiment Type dropdown auto-loads a Markdown template for that type when the body is empty.

Fields

FieldPurpose
Experiment TypeCategorises the entry; drives template and dashboard filters
TC StageStage I–IV or Self Notes for uncategorised entries
Species / AccessionLinks the entry to Registry accessions automatically
Recipe UsedPick a saved recipe; shows as a clickable purple badge on the note
Next Review DateSets a reminder shown in the ⏰ Reminders tab
TagsComma-separated; searchable and filterable

Markdown Toolbar

The editor toolbar wraps selected text or inserts at cursor. All standard Markdown is supported — headings, bold, italic, lists, tables, code blocks, images.

Image Embeds

Click 🖼 to pick an image file. It is base64-encoded and inserted inline — it will be encrypted inside the note. Keep images under 1MB to avoid large file sizes.

Search

Typing in the search box instantly filters the sidebar by title, species, and tags. Pressing Enter triggers a deep search — it decrypts every note body and finds matches with highlighted excerpts.

Filters

The filter pills below the search box filter by experiment type, TC stage, or show only overdue reminders / contamination entries.

Accession Registry

Track every plant accession — species, accession number, origin, source, acquisition date, and status. Linked lab entries appear automatically on the accession page: any note whose Species/Accession field contains the accession's species name is shown as a linked entry.

Status options

  • Active in culture — currently growing in vitro
  • In acclimatization — Stage IV, being hardened off
  • Dormant — paused, not actively subcultured
  • Transferred out — moved to another lab or person
  • Lost — culture failed or contaminated

Media Recipe Book

Store complete media formulations. Each recipe records base medium, sucrose, agar/gelrite, pH, growth regulators, autoclave parameters, and free-text preparation notes.

Base Medium

Choose from the curated list covering general (MS, LS), orchid-specific (Knudson C, VW, Pierik), monocot/banana (B5, N6, MS-Musa), and woody plant media (WPM, DKW). Select ✏️ Custom to type any name.

Autoclave

Use the preset buttons (121°C/15m, 121°C/20m, 121°C/45m for large autoclaves, 115°C/20m) or type exact temperature and hold time manually. For heat-sensitive components, use Filter sterilized (sets temp to 0).

Linking to notes

When creating a lab entry, the Recipe Used dropdown lists all saved recipes. The linked recipe appears as a purple badge on the note — clicking it opens the recipe directly.

Reminders

Any lab entry with a Next Review Date appears in the ⏰ Reminders tab, grouped by urgency:

  • 🔴 Overdue — past the review date
  • 🟡 This week — due within 7 days
  • 📅 This month — due within 30 days
  • 🗓 Later — more than 30 days away

The nav badge shows a count of overdue + this-week entries. Click any reminder card to open the note.

⏱ Tools — Timers & Calculator

Countdown Timers

Click ＋ Add Timer to create a countdown. Enter hours, minutes and seconds, give it a name (e.g. "Autoclave", "Surface sterilization"), then press ▶. Up to 5 timers can run simultaneously. A repeating beep plays when any timer reaches zero. Press ↺ to reset.

Stopwatches

Multiple stopwatches with lap recording. Press ⏱ Lap to record a split time without stopping. Press ↺ to clear.

Regular / Scientific Calculator

Standard calculator with memory. Switch to Scientific for sin/cos/tan and inverse trig (uses DEG by default, toggle to RAD), log, ln, √, x², xʸ, π and e.

BioChemistry Calculator

  • Molarity — calculates molar concentration from MW, mass, and volume
  • Dilution — C₁V₁ = C₂V₂; leave V₁ blank to solve for it
  • % Solution — w/v percentage from mass and volume
  • pH — pH from [H⁺] or from pOH
  • PGR Convert — converts growth regulator concentrations between mg/L and μM using molecular weight

Export & Backup

Click ⬇ Export in the top bar to download all notes as a ZIP of decrypted Markdown files, plus accessions and recipes as JSON. Each note file includes YAML front matter with all metadata.

Your data is also always in the GitHub repo as encrypted .enc files — every save is a git commit, so you have full version history. The repo itself is your backup.

Print / PDF

Open any note or recipe and click 🖨 Print / PDF. This opens a clean print-optimised page. Use your browser's print dialog and choose Save as PDF.

PWA — Install as App

On Chrome/Edge: click ⬇ Install App in the nav bar when it appears (usually after your first visit). On Safari/iOS: tap the Share button → Add to Home Screen. The app then opens in its own window without browser chrome.

The app shell is cached offline — you can open it without internet. However, loading or saving notes requires GitHub API access (internet connection).
